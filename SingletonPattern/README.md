# Singleton Pattern

For when you for some reason need a singleton.
This is thread-safe due to [Magic Statics](http://www.open-std.org/jtc1/sc22/wg21/docs/papers/2008/n2660.htm).

Supported from VC++2015 and GCC C++11.
